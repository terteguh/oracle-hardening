2.2.6 Ensure 'REMOTE_LOGIN_PASSWORDFILE' Is Set to 'NONE' (Automated)
2.2.7 Ensure 'REMOTE_OS_AUTHENT' Is Set to 'FALSE' (Automated) 
2.2.8 Ensure 'REMOTE_OS_ROLES' Is Set to 'FALSE' (Automated)
